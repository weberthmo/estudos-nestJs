  Requisitos

    Instalação do Node
        sudo apt-get install -y nodejs
        https://nodejs.org/dist/v10.15.1/node-v10.15.1-x64.msi

    Instalação do NestJS
        npm i -g @nestjs/cli
        npm i -g @nestjs/cli@6.9.0
        nest --version
Criando projeto
    nest new nome-do-projeto

Separar arquivos em dominios
    usuarios\src\usuario

Validação de informações inviadas por clientes
    yarn add class-transformer@0.2.3 class-validator@0.12.2
